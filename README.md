# Ethio-Grog-calander-converter

A highly accurate, cross-platform desktop application built with Python and CustomTkinter that converts calendar dates smoothly between the Gregorian Calendar (G.C.) and the Ethiopian Calendar (E.C.) tracks.

## Features
- **Ethio to Grog converter:** Convert instantly from Gregorian ➔ Ethiopian or Ethiopian ➔ Gregorian.
- **Leap Year Calculations:** Programmatically tracks offsets for standard Gregorian leap anomalies alongside the 4-year cycle of the 13th month (*Pagume*).
- **Holiday Engine:** Identifies fixed Ethiopian calendar holidays (e.g., Enkutatash, Meskel, Genna) automatically upon conversion.
- **Modern Adaptive Theme UI:** Features an integrated dynamic layout supporting automatic light and dark styling tracks.
- **Input Guard & Validation:** Includes real-time string checking preventing execution anomalies from invalid monthly bounds.
